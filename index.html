A-Pc

A new pc builder

In dit project was de opdracht om een pc builder te maken met hierin een soort pricewatch. Eingelijk moest het een combinatie worden van de pc builder van Alternate en de pricewatch van Tweakers. De bedoeling was dat de product informatie van verschillende webshops moest worden gecrawld en dit moest worden opgeslagen in een database, in ons geval document bassed MongoDB. Met deze data moest de gebruiker een pc kunnen samenstellen en het systeem moet controleren op compabiliteit van de componenten. Deze opdracht is gerealiseerd in Python met het web framework Django

Bekijk hier het product A-Pc.

  • Categorie: Web
  • Vaardigheden: Python Django Linux Scrapy Bootstrap
  • Leerjaar: 2 periode 1 en 2
  • Datum: September 2014
